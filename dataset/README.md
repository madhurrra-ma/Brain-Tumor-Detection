# Brain Tumor MRI Dataset

The dataset used in this project is **not included** in the repository due to its large size.

## Download

[Download Dataset](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset)

## Folder Structure

After downloading and extracting the dataset, organize it as follows:

```text
dataset/
├── Training/
│   ├── glioma/
│   ├── meningioma/
│   ├── notumor/
│   └── pituitary/
│
└── Testing/
    ├── glioma/
    ├── meningioma/
    ├── notumor/
    └── pituitary/
```

## Note

Ensure that the `Training` and `Testing` folders are placed inside the `dataset` directory before running the notebook.

If your dataset is stored in a different location, update the dataset path in the notebook accordingly.
