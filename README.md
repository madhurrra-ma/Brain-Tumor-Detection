# 🧠 Brain Tumor Detection using Deep Learning

### MRI Brain Tumor Classification using Transfer Learning and TensorFlow



A deep learning-based system for classifying **Brain MRI images** into multiple tumor categories using **Deep Learning**. The project explores and compares multiple state-of-the-art CNN architectures to achieve accurate brain tumor classification.

# Overview

Brain tumors are among the most critical neurological disorders where early diagnosis can significantly improve treatment outcomes.

This project develops an automated brain tumor classification system using **Deep Learning** on MRI brain scans. Multiple pretrained convolutional neural networks are fine-tuned and evaluated to classify MRI images into different tumor categories.

---

# Features

- Multi-class Brain Tumor Classification
- Transfer Learning using pretrained CNNs
- Data Augmentation
- Fine-Tuning of pretrained models
- Confusion Matrix & Classification Report

- Comparative evaluation of multiple architectures

# How to Use

#### 1. Requirements:

Ensure you have the necessary libraries and dependencies installed. You can find the list of required packages in the requirements.txt file.

#### 2. Download Data:

Download the Brain Tumor MRI Dataset from Kaggle mentioned in the dataset section of the project.

#### 3. Run the Jupyter Notebook:

Open the provided Jupyter Notebook file and run each cell sequentially. Make sure to update any file paths or configurations as needed for your environment.

#### 4. Training and Evaluation:

Train the models using the provided data and evaluate their performance using metrics such as accuracy and loss.

#### 5. Interpret Results:

Analyze the model's performance using the visualizations and metrics provided in the notebook.
